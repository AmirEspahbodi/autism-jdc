================================================================================
USAGE EXAMPLES
================================================================================

CLI:
  python main.py --mode full              # Train + evaluate
  python main.py --mode train --epochs 5  # Training only
  python main.py --mode eval              # Evaluation only
  python main.py --model mistral          # Use Mistral 7B

Python API:
  from config import SystemConfig
  from infrastructure import LoRAAdapter, MockDataLoader
  from application import FineTuneModelUseCase

  config = SystemConfig()
  trainer = LoRAAdapter(config)
  data_loader = MockDataLoader()

  use_case = FineTuneModelUseCase(trainer, data_loader, config)
  use_case.execute()

CUSTOMIZATION

  Edit config.py or use --model flag

Adjust Hyperparameters:
  python main.py --epochs 10 --batch-size 8 --learning-rate 1e-4

Use Real Data:
  Replace MockDataLoader with FileBasedDataLoader in main.py

Modify LoRA Settings:
  Edit config.py â†’ LoRAConfig class

TROUBLESHOOTING
================================================================================

Out of Memory:
  python main.py --batch-size 2
  # Or reduce LoRA rank in config.py

Slow Training (No GPU):
  Use Google Colab with GPU
  Or reduce training examples

  pip install -r requirements.txt --force-reinstall
